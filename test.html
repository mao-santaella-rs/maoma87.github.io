Lists

Unordered List

This is a paragraph before an unnumbered list (ul). Note that the spacing between a paragraph and a list before or after that is hard to tune in a user style sheet. You can't guess which paragraphs are logically related to a list, e.g. as a "list header".

  • List Item 1
  • List Item 2
  • List Item 3 - Well, probably this list item should be longer. Note that for short items lists look better if they are compactly presented, whereas for long items, it would be better to have more vertical spacing between items.
  • List Item 4 - This is the last item in this list. Let us terminate the list now without making any more fuss about it.
